# Laurindo Dumba - Portfólio de Engenheiro de Dados

Um portfólio moderno e responsivo que apresenta experiência profissional, projetos e habilidades em Engenharia de Dados e Arquitetura em Nuvem.

## Recursos

- **Design Responsivo** - Funciona perfeitamente em desktop, tablet e dispositivos móveis
- **UI/UX Moderno** - Tema escuro com gradientes de acentuação e animações suaves
- **Desempenho Rápido** - CSS e JavaScript otimizados para carregamento rápido
- **Navegação Suave** - Cabeçalho fixo com comportamento de rolagem suave
- **Vitrine de Projetos** - Com mais de 5 projetos de engenharia de dados
- **Integração Social** - Links para LinkedIn, GitHub, Medium e Lattes

## Estrutura do Projeto

```
Portfolio/
├── portfolio.html      # Página principal do portfólio
├── eventos.html        # Página de eventos
├── main.css           # Folha de estilos (arquivo separado opcional)
├── foto/              # Pasta de imagens
│   └── foto.png       # Foto de perfil
└── README.md          # Este arquivo
```

## Design

- **Paleta de Cores**: Fundo escuro (#0f0f0f) com gradientes ciano/azul (#00d4ff, #0099ff)
- **Tipografia**: Fontes do sistema (Segoe UI, Roboto)
- **Animações**: Efeitos de fade-in, slide-up e fill-bar para interesse visual

## Seções
